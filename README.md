# USB-threat-monitor

This is a basic cybersecurity project that monitors USB activity and detects potentially risky files.

## Features

- Detects USB insertion and removal with timestamps  
- Lists all files inside the USB  
- Detects executable files (.exe)  
- Displays warning alerts for suspicious files  

## Technologies Used

- Python  
- OS Module  
- Datetime Module  

## Purpose

This project was built to understand real-world cybersecurity concepts like:
- Device monitoring  
- Threat detection  
- Basic digital forensics
